# CPU Scheduling Algorithm Simulator 🖥️⏳

A comprehensive performance analyzer for core Operating System scheduling algorithms, designed to compare efficiency across different workloads.

## ✨ Supported Algorithms
- **FCFS** (First-Come, First-Served)
- **SJF** (Shortest Job First - Non-preemptive)
- **SRT** (Shortest Remaining Time - Preemptive)
- **Round Robin** (with configurable time quantum)
- **Priority Preemptive**

## 🛠️ Tech Stack
- **Language:** Python
- **Visualization:** Matplotlib (Automated **Gantt Chart** generation).
- **Analysis:** Calculates Average Waiting Time and Turnaround Time for performance comparison.

## 🚀 Performance Metrics
The system outputs a detailed process table and a visual timeline (Gantt Chart) for each algorithm to show exactly how the CPU handles process switching.
